### Реализация алгоритма фазового вокодера. 

Фазовый вокодер используется для растяжения сигнала по времени без изменения высоты тона и устранения эффектов, 
появляющихся после применения оконного преобразования Фурье (настройка когерентности фаз). 

В дальнейшем, это можно использовать в задаче `pitch shift`, позволяя создавать вокальные эффекты путем изменения 
высоты звка: для этого достаточно, например, растянуть сигнал в 2 раза, и потом проиграть его в 2 раза быстрее. 
Тогда, частоты будут удвоены, и pitch будет смещен.


Описание алгоритма фазового вокодера доступно по [ссылке](https://www.ee.columbia.edu/~dpwe/e6820/papers/FlanG66.pdf).
